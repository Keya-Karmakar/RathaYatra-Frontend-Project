# 🚩 Ratha Yatra Celebration Website

A simple and interactive **Ratha Yatra Festival** website created using **HTML, CSS, and JavaScript**.

The project celebrates the sacred journey of **Lord Jagannath, Lord Balabhadra, and Devi Subhadra** with a moving Rath animation, blessings, and a Masi Bari page.

---

## 🌟 Features

- 🛕 Beautiful Ratha Yatra themed homepage
- 🚩 Animated Rath movement
- 🙏 "Jai Jagannath" greeting
- 💬 Blessing alert message
- 🏵️ Automatic navigation to Masi Bari page
- 🖼️ Masi Bari image page
- 📱 Responsive design
- 🎨 Attractive UI using CSS

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 How to Run

1. Download or clone the repository.

```
git clone https://github.com/your-username/RathaYatra.git
```

2. Open the project folder.

3. Open **index.html** in any modern web browser.

No installation is required.

## 🌸 Festival Significance

Ratha Yatra is one of the most celebrated festivals of Lord Jagannath. During this festival, Lord Jagannath, Lord Balabhadra, and Devi Subhadra travel from the Jagannath Temple to the Gundicha Temple (known as **Masi Bari**) on beautifully decorated chariots.
